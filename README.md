## Installation And Usage

To run this project run

```bash
  https://github.com/sagar-demo/Album-Collection
```

Go to directory

```bash
  cd album_list
```

Install all dependecies

```bash
 npm install
```

Run Project

```bash
  npm start
```

Now you can access on localhost:3000
